üîß Troubleshooting

Port 3002 Already in Use

If you see "port 3002 already in use", either:

  • Kill the process: Press Ctrl+C twice
  • Use a different port: npm run dev -- --port 3003

npm install Fails

Try clearing the cache:

npm cache clean --force npm install

Tests Won't Run

Clear cache and reinstall:

rm -rf node_modules package-lock.json npm install npm test

Module Not Found Errors

Make sure Dependencies are installed:

npm install

TypeScript Errors

Check types with:

npm run type-check
